
Appleseed Roadmap

Updated:  06-27-2010
Version:  0.7.4

0.7 SERIES
----------
Base features, API, framework, themability, internationalisation.

0.7.4  - MVC framework, Internationalisation, Configuration inheritance, New theme, SHA512 for password storage
0.7.5  - Refactor legacy code, Html form functions, Refactor CRUD functions, Integrate JQuery, Administrator framework and front end refactoring.
0.7.6  - Add micro-blogging and "wall" components, round out features and bug fix friends, messaging, and login components.
0.7.7  - Add user options and system logging components, round out features and bug fix photos, journals, and comments components.
0.7.8  - Add newsfeed component, round out features and bug fix groups, pages, and articles components.
0.7.9  - Add search and events components, build out JQuery integration and framework
0.7.10 - Add chat, video, and music components
0.7.11 - Refactor and round out new components.  Semi-Freeze protocol and API with this release.

0.8 SERIES
----------
Optimization, Standards, Security, Connectivity and Compatibility.  

- Finalization of a standard, open protocol for communication between Appleseed sites and other open source projects
- Compatibility with Flickr, Twitter, and other major services (via newsfeed hooks).
- Profile importers for Facebook, MySpace, and others.
- Speed and performance optimization: server, browser, and network.
- Full audits for sensible defaults and system security.

0.9 SERIES
----------
FEATURE FREEZE!  Bug testing, Code Cleanup, Documentation

1.0 SERIES
----------
API Enhancement, Scalability, User Interface Refinement, Security Fixes, and Optimization.
